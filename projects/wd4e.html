    • Web Design for Everybody

    Timeline: October 2022 - Present

    Role: Instructional Aide

    Supervised by: Colleen Van Lent

    TL;DR: Support the Web Design for Everybody Coursera course by developing all HTML, CSS, and JavaScript resources, including the course website.

    Skills: HTML, CSS, JavaScript, Git

    About

    Following my high achievement in Colleen van Lent's Web Design, Development, and Accessibility course, I was hired on to support a complete overhaul of her Web Design for Everybody Coursera course. Colleen's first iteration of Web Design for Everybody, which was created 10 years ago, has seen great success, with roughly 300,000 enrolled students. During the course remake, I have been responsible for generating the course material and quizzes and developing an interactive course website.

    Course concepts include: intro to HTML, intro to CSS, intro to JavaScript, responsive design, and web accessibility. The finished course is set release in November, 2023.
